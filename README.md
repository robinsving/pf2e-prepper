# PF2e Prepper - User Documentation

## Overview
The PF2e Prepper module enhances the Pathfinder 2e system in Foundry VTT by allowing prepared spellcaster classes to save and load multiple spell configurations. This feature is particularly useful for characters who need different spell loadouts for various situations such as combat, exploration, or social encounters.

## Installation
1. In Foundry VTT, navigate to the "Add-on Modules" tab
2. Click "Install Module"
3. In the "Manifest URL" field, paste: `https://github.com/robinsving/pf2e-prepper/releases/latest/download/module.json`
4. Click "Install"
5. Enable the module in your world

## Features
- Save multiple spell configurations for Prepared and Flexible spellcaster classes
- Name and describe each spell list for easy reference
- Quickly load a stored spell list (during daily preparation)
- Manage (rename, update, delete) your saved spell lists
- Supports multiclass characters with multiple spellcasting entries

## Usage

### Accessing the Spell Lists Manager
1. Open a character sheet for a prepared spellcaster
2. Navigate to the "Spellcasting" tab
3. Click the "Manage Spell Lists" button in the tab header

### Creating a New Spell List
1. Prepare your spells as normal through the character sheet
2. Open the Spell Lists Manager
3. Click the "New List" button
4. Enter a name and optional description for your spell list
5. Click "Save"

### Loading a Spell List
1. Open the Spell Lists Manager
2. Click on the tab for the spell list you want to load
3. Click the "Load" button
4. Confirm that you want to replace your current prepared spells
5. Your character's prepared spells will be updated to match the loaded list

### Updating a Spell List
1. Open the Spell Lists Manager
2. Click on the tab for the spell list you want to update
3. Click the "Save" button to update the list with your current spell preparation

### Renaming a Spell List
1. Open the Spell Lists Manager
2. Click on the tab for the spell list you want to rename
3. Click the "Rename" button
4. Enter a new name and optional description
5. Click "Save"

### Deleting a Spell List
1. Open the Spell Lists Manager
2. Click on the tab for the spell list you want to delete
3. Click the "Delete" button
4. Confirm that you want to delete the list

## Tips and Best Practices
- Create themed spell lists for different situations (combat, exploration, social)
- Name your lists clearly to easily identify their purpose
- Update your lists when you learn new spells that would be useful in specific situations
- For multiclass characters, saved lists will include spells from all prepared spellcasting entries

## Troubleshooting
- If the "Manage Spell Lists" button doesn't appear, ensure your character has a prepared spellcasting entry
- If loading a spell list doesn't work, the character may no longer have access to some of the spells in the list
- If you encounter any issues, check the browser console for error messages

## Compatibility
- Requires Foundry VTT v11 or later
- Requires Pathfinder 2e system v5.0.0 or later
- Compatible with most other Pathfinder 2e modules
